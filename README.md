## 📝 Capstone 
##### VR, 인공지능을 활용한 던전 RPG 제작 및 웹 서비스 제공

----

## 웹 기능
- 회원가입
- 로그인, 로그아웃
- JWT토큰 기반 로그인, 로그아웃
- 게시물 CRUD
- 게시물 비공개 설정
- 게시물 이미지 첨부
- 검색 기능
- 댓글 기능
- 내 정보 수정
- 비밀번호 찾기
- 회원탈퇴
